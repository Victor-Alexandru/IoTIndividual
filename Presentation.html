Overview
The police29 project helps the user capture photos every five seconds without opening the camera ,and without pushing any button. The photos are updated to the cloud (firebase storage) and are accesed by a web application that display the last 4 photos taken.

Pre-requisits
List of needed components :

  • 1.Smartphone (either Ios or Android (with sdk+21) - the app is non native made in flutter)
  • 2.Browser - to display the web page
Setup and Build
  • 1. Create a Firebase project on the console here https://firebase.google.com/
  • 2.Add an android app to your project and setup firebase to the flutter application (download google.json file)
  • 3.Use the folowing plugins in the flutter app : camera, path_provider, path, firebase_storage
  • 4.Create a web app and add the config keys to get access to the firebase storage
    IMPORTANT STEP : DOWNLOAD A GENERATE KEY TO HAVE READ ACCESS TO THE STORAGE
  • Firebase rules set to default :
